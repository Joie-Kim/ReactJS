# React

## 🤔 Why

- 프론트엔드에 대한 이해를 위해<br>
  : 지난 프로젝트에서 프론트엔드 경험이 없는 것에 아쉬움을 느꼈습니다. ( [📎 지난 프로젝트 회고](https://joie-kim.github.io/Review-Team-Project/) )
- 인스타그램 클론 코딩을 위해<br>
  : 프론트엔드, 백엔드를 모두 경험해 보기 위한 개인 프로젝트로 인스타그램을 클론 코딩을 할 것이며,<br>
    ReactJS에서 데이터를 다루는 것을 익혀 이를 React-native로 적용할 예정입니다.
