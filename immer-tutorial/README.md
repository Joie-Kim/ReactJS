### 12장 immer 사용하여 더 쉽게 불변성 유지하기
- immer 사용법 익히기

전개 연산자와 배열의 내장 함수를 사용하면 간단하게 배열 혹은 객체를 복사하고 새로운 값을 덮어 쓸 수 있다.<br>
하지만 객체의 구조가 엄청나게 깊어지면 불면성을 유지하면서 이를 업데이트 하는 것이 매우 힘들다. (값 하나를 업데이트 하는 데에 코드를 열 줄 넘게 작성 해야 할 수도... 🥲)
